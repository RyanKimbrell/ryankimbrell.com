SPOTIFY TOOLS

http://spotifytools.net is a web application that I developed by myself in order to provide Spotify users with access to data about the music they love to listen to, and to create big playlists based off of user-seeded recommendations. It utilizes Python/Django with SQL on the backend, and is served on the front-end using a combination of Django and JavaScript, and styled with a combination of my own custom CSS and bootstrap. Interacts and organizes data pulled from Spotify's Web API and uses OAuth to gather user credentials. (Click Images To Enlarge)
